# Demo Video APP

A fully functional flutter application that allows the end-user to upload video, thumbnail, title, and description which then gets added to the list of videos on the home screen, where the user can see their videos and videos uploaded by other users. This app also allows the user to delete their uploaded files.

This application uses Google Firebase as its backend for user authentication, storage, and storing metadata. It is using Chewie package for streaming files from the Firebase Storage.

## Usage

This app can be used for:

- Video Journal
- Sharing recipes
- Complaint app, where user can record any issue and share it with others
- Tutorials/E-learning app

## Packages used

This application uses the following packages from [pub.dev](https://pub.dev/)

- [firebase_core: ^1.1.1](https://pub.dev/packages/firebase_core)

- [cloud_firestore: ^2.1.0](https://pub.dev/packages/cloud_firestore)
- [google_sign_in: ^5.0.2](https://pub.dev/packages/google_sign_in)
